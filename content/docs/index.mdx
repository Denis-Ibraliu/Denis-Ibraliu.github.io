## Open Source Libs

This repository is an Nx monorepo for open-source libraries created during day-to-day work.

Some packages begin as small utilities or experiments that solve a real production problem, then graduate into reusable libraries here. The repo currently has one published package and is meant to keep growing as more packages are extracted over time.

## Available now

- [`@commonkit/conditional`](./packages/conditional) for declarative conditional rendering in React
